# cself

`cself` is a cli for the `checksum` package.

## usage

```
ยง cself -h
NAME:
   cself - generate hashes for files or stdin

USAGE:
   cself [OPTIONS] file

COMMANDS:
   list-hashes, l  list hashes

GLOBAL OPTIONS:
   --list-hashes value  list hash algorithms
   --help, -h           show help (default: false)

```
